## Usage

### Step 1

Add maven in build.gradle or settings.gradle.

```groovy
    repositories {
        maven { url "https://raw.githubusercontent.com/ubxsamples/usdk/main" }
    }
```

### Step 2

Add dependencies in build.gradle.
```groovy
    dependencies {
       implementation('com.ubx:usdk:1.4')
    }
```
